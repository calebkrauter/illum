# SCOPE

* The primary aim of this proejct is to learn frontend and backend development in TypeScript React using CSS, Next.js and to create a deployment pipeline. THe hope is to grow in experience, knowledge and capability.

* The secondary aim and tangable goal of this project is to display my experience in a meaninful and palatable way for perspective employers to learn about me, abilities and get a sense of who I am.

### Features

* Frontend
  a. Experience/Project Cards
  b. Intuitive Navigation
  c. Uniquely stylized and intuitive design
  d. Theme Switcher that sets a cookie
* Backend
  a. Serves content with SSR to improve SEO
  b. Light/Dark mode set with cookie
  c. Potentially a way to update the content from client to the backend

### Deployment
Create an automated pipeline to make deployment quick and painless. Start with Vercel, consider other options for learning experiences.
